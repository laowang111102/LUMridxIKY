# 停车场管理系统 java1021

本项目是基于 SpringBoot 和 Vue 的停车场管理系统，采用前后端分离的架构设计。

## 一、项目概述

本项目旨在为停车场提供一套全面的管理解决方案，支持不同角色的用户进行个性化操作。

## 二、技术栈

- **支持环境**：JDK 1.8
- **数据库**：MySql 5.7
- **开发软件**：MySql、Idea

## 三、功能模块

### 1. 个人信息管理

- 新增用户信息
- 修改用户信息
- 删除用户信息
- 查看用户信息

### 2. 预定停车位

- 预定停车位

### 3. 缴费信息

- 查看个人余额
- 充值
- 缴费

### 4. 车位信息管理

- 新增车位信息
- 修改车位信息

### 5. 停泊车辆管理

- 新增用户停车信息
- 修改用户停车信息

### 6. 车辆进出管理

- 新增车辆进入信息
- 修改车辆驶出信息
- 缴费

### 7. 登录日志查询

- 查询用户登录日志

## 四、系统角色

系统分为两个角色：

- 管理员
- 普通用户

不同角色具备不同的功能权限。

## 五、运行环境

- JDK 1.8 或以上版本
- MySql 5.7 或以上版本

## 六、部署步骤

1. 准备开发环境（JDK 1.8，MySql 5.7）
2. 使用 Idea 导入项目
3. 初始化数据库
4. 运行项目

## 七、目录结构

```
- src
  - main
    - java
    - resources
    - webapp
- doc
- README.md
```

## 八、核心亮点

- 前后端分离，便于维护和扩展
- 支持不同角色登录，功能权限分明
- 完善的停车管理功能，提高停车场运营效率
- 易于部署和运行，支持主流开发工具和数据库
